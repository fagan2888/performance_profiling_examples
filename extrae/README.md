# Extrae Trace File

To explore the Extrae trace file, download the Paraver visualisation application from the [BSC
tools website](https://tools.bsc.es/paraver).  

Open the Paraver GUI, then choose `File->Open Trace` and load the file `epoch_8rank.prv`.  Then
click the button directly below the File menu labelled "New single timeline window" to open the
timeline view.

The BSC tools website has [Paraver Tutorials](https://tools.bsc.es/paraver-tutorials) available.
